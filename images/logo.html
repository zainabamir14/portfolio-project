HTML builds the structure of web pages and organizes content for websites.

CSS styles web pages by controlling layout, colors, fonts, and design, making websites look attractive and user-friendly.

JavaScript adds interactivity and dynamic behavior to web pages, like animations, form validation, and real-time updates.





jQuery simplifies JavaScript tasks like event handling, animations, and AJAX, making web development faster and easier.

Bootstrap helps build responsive, mobile-friendly websites quickly using pre-designed HTML, CSS, and JavaScript components.

PHP in web development is used for building dynamic websites, handling server-side logic, processing forms, and connecting to databases like MySQL.
